Dans Ardour, le travail s'effectue dans l'une de ces quatre fenêtres : L'Editor, La Table de mixage, L'Enregistreur, ou la Grille de Cue.

L'Éditeur, la table de Mixage, l'Enregistreur, et le Lanceur de Clip partagent la même barre d'outils (le haut de la fenêtre). Les sections affichées dans la barre d'outils peuvent être customisées pour le type de travail de l'utilisateur dans Préférences > Apparence > Barre d'outils.

Pour passer à l'un des quatre mode :

  • avec le Bouton de sélection de mode dans le coin supérieur droit
  • avec le menu Fenêtre > Éditeur (or Console de Mixage / Enregistreur / Grille de Cue) > Afficher.

Le raccourcis M permet de passer de l'Éditeur à la Console de Mixage.

Plusieurs fenêtres peuvent être visible en même temps (par exemple pour une config multi-écran) en utilisant le sous-menu Fenêtre > Editeur (or Console de Mixage / Enregistreur / Grille de Cue)> Détacher

L'Éditeur

La fenêtre Éditeur inclus le support des pistes d'édition, où l'on va disposer les données audio et MIDI selon la ligne temporelle. C'est dans cette fenêtre que l'on fait l'édition et l'arrangement du projet. La fenêtre a une logique "horizontale" : la ligne temporelle va de gauche à droite, la tête de lecture qui indique la position actuelle dans la session se déplace de gauche à droite. La fenêtre représente le temps d'une manière très litérale.

Il est possible d'afficher un bandeau de canal (Entrée ou Bus de Sortie) dans la fenêtre d'édition et certains pensent suffisant de travailler le mixage sans ouvrir la Fenêtre de Console. Mais globalement ces deux fenêtres sont nécessaires selon les étapes du travail

La Console de Mixage

La Console de Mixage représente le parcours du signal et sera certainement la fenêtre utilisée lors d'une session de mixage. Elle inclus les bandeaux de canal pour chaque pistes et bus de la session. Elle a une logique "verticale" : le signal entre en haut, passe par les traitements avant d'attendre les sorties en bas.

Pour approfondir le processus du mixage, voir Mixage.

L'Enregistreur

La fenêtre Enregistreur est une représentation très spécialisé qui propose une vue compacte du status d'enregistrement et d'écoute de toutes les pistes, ainsi qu'une ligne temporelle simplifiée permettant d'avoir une vue complète. La plupart des informations de ce mode sont disponibles dans les modes Console de Mixage et Éditeur mais l'Enregistreur vise à avoir une vue synthétique pour garder le controle lors d'une session d'enregistrement

L'Enregistreur est détaillé dans sa page.

The Cue Grid

The Cue Grid window, unlike the rest of Ardour, allows for a non-linear workflow. It is a clip launcher, that allows to chain and combine various loops and samples, and program events.

Instead of anchoring these samples or loops on a timeline, the Cue Grid gives them instruction on when they are triggered, how they are played, what happens at the end of the clip, etc…

The Cue section of this manual describes this worfkow.
