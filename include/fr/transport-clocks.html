Les horloges dans Ardour, sont utilisées pour afficher précisément le temps. Dans de nombreuses situations, c'est une des manière de changer la vakeur du temps.

Dans la barre de transport de la fneêtre de montage, par défaut il y a deux horloges qui affichent la position du lecteur et d'autres informations concernant le transport et la ligne temporelle. On les appelle les horloges de transport; celle de gauche est l'horloge primaire de transport (elle affiche toujours la position du lecteur) et celle de droite est l'horloge secondaire du transoprt.

Le fait d'avoir deux horloges permet de connaitre la position du lecteur selon deux unités de temps différentes sans avoir à changer de réglages. Par exemple, on peut afficher le temps à la fois en Code Temporel et Mesures/Temps. Si l'on souhaite, il est possible de cacher l'horloge secondaire grâce aux Préférences, dans Apparence > Barre d'outils > Afficher l'orloge secondaire.

Dans Ardour, on peut modifier toutes les horloges de la même manière. Allez voir Editing Clocks pour savoir comment. Lorsque vous modifiez le temps dans l'horloge de transport, le lecteur va se re-positionner de la même manière qu'avec d'autres opérations.

Les horloges de transport ont des attributs spéciaux liés à leur fonction:

Panneau d'Information

Sous chaque horloge, il y a un panneau d'information qui le mode d'horloge actuel:
Mode Information
Code Temporel / Minutes:Secondes / Secondes Source du Code Temporel (INT siginifie qu'Ardour génère son propre code temporel)
Mesures:Temps Tempo et Métrique actuels. Cliquer un de ces boutons permet de changer les valeurs.
Échantillons Fréquence d'échantillonnage (SR pour Sample Rate) et décalage son/image', tels que définis dans les Propriétés de session.

Origine du Temps

Avec le menu contextuel du clic-Droit, il est possible de choisir une nouvelle origine du temps, le point "zéro", parmis les suivants:

Afficher le temps absolu Le point "zéro" sera le début de la ligne temporelle (quelque-soit le début de la session et les décalages de Code Temporel).
Afficher le delta avec le curseur Le point "zéro" est le point d'édition choisi avec le Sélecteur d'édition, par exemple le marqueur sélectionné.
Affiche le delta avec le repère de l'origine Le point "zéro" sera le repère début de la session.
  • L'horloge pourra afficher une valeur négative ou positive selon les positions temporelles respectives du point zéro choisi et du lecteur.
  • Les horloges auront une couleur différente selon qu'elles affichent le mode delta à l'édition ou delta à l'origine afin d'éviter toute confusion.
  • Lorsque l'horloge est dans l'un de ces deux modes, on ne peut pas changer sa valeur.
