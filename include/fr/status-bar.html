La barre de statut est une barre d'information en haut de la fenêtre. Un clic-droit n'importe où dans la barre permet de choisir les informations affichées, via une liste à cocher:
Chemin de la session: Le dossier où se trouvent les fichiers de la session
Nom du cliché et indicateur de modification: Le nom du cliché sur lequel on travaille. Si la session a été modifiée depuis la dernière sauvegarde, les noim sera préfixé d'u astérisque(*).
Travail sur fichier-crête actif: (s'affiche seulement lors de la création du fichier-crête) Affiche le nombre de fichier-crête restant à créer
Format de fichier: Le format de fichier utilisé pour la session, ce qui inclut les enregitrements.
Format du code temporel: c'est le nombre d'image par seconde du code temporel utilisé dans la session (pour les vidéos)
Audio: Affiche la fréquence d'échantillonnage utilisé pour cette session, et la latence calculée pour la taille du tampon
Stockage: Affiche l'espace de stockage libre comme temps d'enregistrement disponible selon les paramètres actuels de la session
DSP: pour "Digital Sound Processing", affiche la charge du système utilisé par Ardour et pour le traitement en temps-réel. Cette valeur en pourcentage, correspond au rapport entre temps de traitement effectif et temps de traitement disponible.
Horloge système: Affiche l'horloge système (utile dans le mode plein-écran)
Voyant journal: Indique si Ardour a rencontré un problème ou une erreur.

Certains de ces indicateurs permettent, grâce à un double-clic dans le champ d'info, d'accéder à des actions précises:

Chemin de la session / Nom du cliché et indicateur de modification Ouvre le dossier de session dans le gestionnaire de fichier du système
Format de fichier: Ouvre la section Media des Propriétés de la Session
Format du code temporel: Ouvre la section Code Temporel des Propriétés de la Session
Audio: Ouvre la fenêtre Réglages Audio et MIDI Setup.
Voyant Journal Le voyant du Journal passe en jaune en cas d'avertissement, et en rouge en cas d'erreur. En cliquant le bouton, on affiche le Journal
