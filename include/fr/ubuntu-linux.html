Ubuntu est la distribution Linux la plus populaire pour les ordinateurs portables et fixes. Il s'appuie sur une entreprise à but non lucratif (Canonical Inc.), une philosophie bien définie et une base utilisateurs énorme. C'est donc une plateforme très courante parmis les utilisateurs d'Ardour et autres outils de création et de travail audio professionnels.

Recommandations importantes pour les utilisateurs d'Ubuntu

Pour l'instant, une installation d'outils audio-numérique sur une Ubuntu standard, requiert quelques adaptations de la configuration pour que l'utilisateur puisse avoir les privilèges nécessaires pour les traitements en temps réels (voir ci-dessous). Ubuntustudio, une des variantes officielles qui partage donc les mêmes dépôts qu'Ubuntu, est déjà configuré. D'autres distributions, telles que KXStudio et Dreamstudio sont très principalement basées sur Ubuntu, et comme Ubuntustudio ont ces réglages déjà configurés mais contiennent aussi des versions customs de paquets Ubuntu, souvent plus récents.

Installer Ardour

Il existe des différences et même des bugs dans les paquets natif d'Ubuntu, à cause des différentes méthodes utilisées lors de la compilation. C'est pour cela que les développeurs d'Ardour préconisent l'installation de la version prêt-à-l'emploi officielle d'Ardour que vous pouvez télécharger depuis ardour.org, parce qu'il n'y a pas de support pour les paquets Ubuntu sur le forum ou les autres canaux officiels d'Ardour.

Pour l'installation de la dernière version d'Ardour, suivez les étapes suivantes:

  1. Téléchargez la dernière version publiée depuis ardour.org.
  2. Faites un clic-Droitsur le fichieret choisissez "Propriétés".
  3. Cliquez sur l'ongle "Permissions" et cochez l'option "Allow this file to run as a program".
  4. Fermez le Dialogue et double-cliquez sur le fichier.
  5. Suivez les affichages.

Problèmes liés à la configuration de JACK

Quel problème?

Pour fonctionner comme prévu, JACK a besoin d'un accès à deux fonctions du système d'exploitation, qui sont la Gestion Temps-réel et le verrouillage de la mémoire. Ce qui implique que l'utilisateur qui démarre JACK puisse avoir accès à ces fonctionnalités. Par défaut, Ubuntu crée un groupe utilisateur qui a cette permission mais il n'y ajoute pas les nouveaux utilisateurs par défaut. Lisez here pour en savoir plus. Donc les utilisateurs ne pourront pas utiliser JACK correctement.

Symptomes

Un message du type Verrouillage de la mémoire impossible peut apparaitre au démarrage de JACK. Il est possible que cet avertissement ne soit affiché que dans la fenêtre de message de QTJackCtl (alias JACK Control), donc vérifiez cette fenêtre.

Comment résoudre

Assurez-vous que le fichier /etc/security/limits.d/audio.conf existe. S'il se nomme /etc/security/limits.d/audio.conf.disabled, alors renommez-le comme le premier nom. Dans un terminal, saisissez la commande suivante:

sudo usermod -a -G audio VOTRE-NOM-UTILISATEUR

Ensuite, fermez la session et reconnectez-vous. Dans Ubuntustudio, les utilisateurs sont membres du groupe audio mais pas dans les autres versions officielles.

Signaler un problème
